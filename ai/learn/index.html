Learn the map before collecting tools.

Most people learn AI backwards: first the tool, then the prompt tricks, then a vague sense that maybe they missed the foundations. This section exists to reverse that order.

What belongs here Mental models, distinctions, core concepts, and the logic needed before chasing yet another workflow video.

Layer 1: concepts

Models, context windows, retrieval, agents, automation, and the difference between these terms when they leave marketing language.

Layer 2: judgment

How to tell if a use case is actually a fit, whether a process needs automation, and where human review is non-negotiable.

Layer 3: implementation

Only after the first two layers: tools, systems, orchestration, interfaces, handoff, and maintenance costs.
