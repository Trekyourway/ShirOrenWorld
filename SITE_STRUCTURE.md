# Shir Oren — Site Structure

## Core rule
`/ai`, `/coaching`, and `/treks` are the actual main entry pages for each world.
There is no separate world gateway URL.

## Notes
- Each world owns its own content types.
- Content types are not shared globally across worlds.
- This structure is designed to scale without changing URLs later.
